Proyectos

Proyecto 1: Aplicación Web de Gestión de Usuarios.

Desarrollé una aplicación web que permite a los usuarios registrarse e iniciar sesión utilizando un sistema de autenticación robusto. Utilizando Flask como framework principal y Python como lenguaje de programación, implementé una base de datos en la nube con MongoDB para almacenar la información de los usuarios de manera segura y accesible. La aplicación incluye un sistema de recuperación de contraseña que utiliza SendGrid, lo que permite a los usuarios restablecer sus contraseñas fácilmente a través de un enlace enviado a su correo electrónico. Además, implementé la opción de iniciar sesión mediante Google, lo que simplifica el proceso de autenticación al permitir a los usuarios acceder con sus credenciales existentes. Al registrarse o iniciar sesión con Google, se crea automáticamente un perfil para el usuario, capturando su nombre y proporcionando una experiencia personalizada desde el primer momento. Este enfoque no solo mejora la seguridad, sino que también facilita una interacción más fluida y amigable con la aplicación.

Proyecto 2: Página Web Educativa sobre Ecología.

Desarrollé una página web durante mi etapa en la preparatoria, diseñada para brindar información valiosa sobre la asignatura de Ecología. La plataforma cuenta con un menú de navegación intuitivo que permite a los usuarios acceder fácilmente a diferentes secciones, incluyendo información relevante sobre la preparatoria y un apartado dedicado a dejar comentarios, fomentando así la interacción y el feedback de los visitantes. Además, integré un juego en C++ que funciona como un ahorcado básico, proporcionando una manera divertida y educativa de aprender sobre conceptos ecológicos. La página está diseñada con un enfoque profesional, asegurando que la presentación sea atractiva y accesible para todos los usuarios interesados en el tema. Este proyecto no solo refleja mis habilidades técnicas en desarrollo web y programación, sino también mi compromiso por promover el aprendizaje de la Ecología de una manera interactiva y atractiva.

Proyecto 3: Interfaz Gráfica Web para Escaneo de Puertos Nmap.

Desarrollé una interfaz gráfica web para la herramienta de escaneo de puertos Nmap, integrando un panel administrativo para la gestión de las operaciones. Esta aplicación permite a los usuarios realizar escaneos de red de manera intuitiva a través de una interfaz amigable. Además, incorporé una terminal interactiva basada en `bspwm` de Linux, ofreciendo a los usuarios avanzados la flexibilidad de ejecutar comandos directamente desde el navegador. Este proyecto combina la facilidad de uso de una interfaz gráfica con la potencia de herramientas de línea de comandos, facilitando las tareas de seguridad y administración de red.
